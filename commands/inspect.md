# Unity Inspect Command

Use this command to analyze a specific Unity file.

```bash
bun ${CLAUDE_PLUGIN_ROOT}/unity-yaml/dist/cli.js inspect $ARGUMENTS
```

## Examples

- `/unity:inspect Assets/Scenes/Main.unity`
- `/unity:inspect Assets/Prefabs/Player.prefab`
